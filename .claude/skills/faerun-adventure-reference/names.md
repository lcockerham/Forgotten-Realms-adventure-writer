# Faerûnian Names

Use these canonical names when creating NPCs. Select names appropriate to the region and culture of your adventure.

## Naming Conventions

- **"sar"** (Tharsult, Tashalar, Border Kingdoms): Means "of the blood of," denotes a famous ancestor (e.g., "Baer sar Thardizar")
- **Patronymics**: Some cultures use "son/daughter of" appellations
- **Occupational roots**: tel = "works at/with", forar = traveler/peddler, helder = guard/warrior, turnskull = digger, turnstone = miller
- **Achievement names**: Rare, used mainly by wizards (e.g., "Dastrin of the Three Thunders")
